# Bloom Filter

## About Bloom Filters
#### A Bloom filter is a probabilistic data structure for fast, memory-efficient membership checks. Internally, it uses multiple hash functions to map elements into positions in a bit array rather than storing the elements themselves. This allows O(1) insertion and lookup while using very little space. Because different elements can map to the same bits, false positives are possible. However, a Bloom filter will never return a false negative.
